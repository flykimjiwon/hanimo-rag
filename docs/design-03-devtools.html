RAG 채팅 & 출처

hanimo-rag의 주요 아키텍처는 어떻게 구성되어 있나요?

hanimo-rag는 주로 다음과 같은 아키텍처로 구성되어 있습니다.

  • FAST API 백엔드: 고성능 비동기 API 처리
  • Vector DB: 문서 임베딩 저장을 위한 ChromaDB 또는 Pinecone
  • LLM 연동: 답변 생성을 위한 OpenAI 또는 로컬 모델
