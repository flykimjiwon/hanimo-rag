hanimo-rag이란?

hanimo-rag는 PostgreSQL 하나로 벡터 검색 + 전문 검색 + 지식 그래프를 모두 처리하는 하이브리드 RAG(Retrieval-Augmented Generation) 엔진입니다. Elasticsearch, Pinecone, Neo4j 같은 별도 서비스 없이 하나의 데이터베이스에서 모든 검색 기능이 동작합니다.

pip install hanimo-rag 한 줄이면 설치 완료. 문서를 업로드하면 자동으로 파싱 → 청킹 → 임베딩 → 저장 → 그래프 구축까지 처리하고, RRF(Reciprocal Rank Fusion) 알고리즘으로 3가지 검색 결과를 융합하여 최고 품질의 검색 결과를 반환합니다.

React 대시보드로 비개발자도 문서 관리, 검색 테스트, 지식 그래프 시각화를 할 수 있으며, App Builder로 코드 없이 커스텀 LLM 챗봇 엔드포인트를 만들 수 있습니다.
