🤖 My App

매출 현황을 3줄로 요약해줘

2024년 매출은 전년 대비 23% 성장했습니다 [1].
주요 성장 동력은 신규 고객 확보와 평균 객단가 상승입니다 [2].
4분기 전망은 긍정적이며 연간 목표 달성이 예상됩니다 [1].

할 일: Apps 탭 → 앱 만들기 → 문서 선택 → 질문 입력 → AI가 문서를 근거로 답변!

🕸️ 보너스: 관계도 보기

👨‍💻 개발자용: 코드로 사용하기

Python 3줄이면 끝

pip install -e ".[dev]" 
from modolrag import ModolRAG

rag = ModolRAG()                          # DB 자동 연결
rag.ingest("보고서.pdf")                   # 문서 넣기
answer = rag.ask("매출 현황은?")           # 질문 → 답변

터미널 명령어

modolrag quickstart자동 설치 (Docker+DB+Ollama)
modolrag serve서버 시작
modolrag ingest report.pdf문서 넣기
modolrag search "검색어"검색
modolrag ask "질문"AI 답변
modolrag status상태 확인

REST API

서버가 실행 중이면 어떤 프로그래밍 언어에서든 HTTP로 호출할 수 있습니다.

# 문서 업로드
curl -X POST http://localhost:8000/api/ingest -F "file=@report.pdf"

# 검색
curl -X POST http://localhost:8000/api/search \
  -H "Content-Type: application/json" \
  -d '{"query": "매출 현황"}'

# AI 답변 (실시간 스트리밍)
curl -X POST http://localhost:8000/api/generate \
  -H "Content-Type: application/json" \
  -d '{"query": "매출을 요약해줘", "stream": true}'

전체 API 문서: http://localhost:8000/docs (Swagger UI)

🔧 문제가 생겼을 때

대시보드가 안 열려요 서버가 실행 중인지 확인: modolrag serve
DB 연결 에러 Docker가 실행 중인지 확인: docker ps
검색 결과가 안 나와요 문서 상태가 "vectorized"인지 Documents 탭에서 확인
AI 답변이 안 돼요 Ollama가 실행 중인지 확인: curl http://localhost:11434
전체 상태 확인 modolrag status 실행
